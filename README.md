# RentNest
RentNest is a property listing marketplace. Here, landlords list their properties, and Tenants rent the properties they want.

### Role
There are three roles in this project. 
<ul>
  <li><strong>Landlord</strong>: Who can list properties, manage rental requests, and properties.</li>
  <li><strong>Tenant</strong>: Who can browse available properties and make a rental request for the property. After paying for the property, the tenant can leave a review.</li>
  <li><strong>Admin</strong>: Who can manage users' status, categories, check users, properties, and rental requests.</li>
</ul>

### Flow
Users can create "TENANT" or "LANDLORD" accounts. They can log in and check their profile. A landlord can list properties. A Tenant can browse all the available properties. They can search, filter, and sort properties.</br>
A Tenant can make a rental request for a property. When the Landlord (property owner) approves the request, the Tenant can pay for the request.</br>
After paying for the request, the property becomes automatically rented, and it won't appear on available projects anymore.</br>
A Landlord can check, update, and delete his/her properties, or change the status of a rental request.</br>
An admin can monitor users, properties, rental requests, and create, update, and delete categories. Admin can "BAN" or "ACTIVE" a user.
